Project Overview

Django E-Shop is a web-based e-commerce platform built using Django, designed to provide a seamless shopping experience for customers while offering efficient management tools for store owners and employees. The platform supports user authentication with three distinct roles: owner, employee, and customer, ensuring secure and role-based access to different functionalities. Customers can browse products, add items to their cart, and complete secure transactions through a streamlined checkout process. Owners and employees can manage products by adding, updating, or removing them, while also tracking orders and updating their statuses.

The platform includes an intuitive admin panel that allows store owners to monitor sales, manage inventory, and analyze business performance. A dynamic theming feature enables users to switch between light and dark modes for an improved user experience. Built using Django and Django REST Framework for the backend, along with Tailwind CSS for the frontend, Django E-Shop is designed to be scalable, secure, and user-friendly. The database is managed using PostgreSQL or SQLite, depending on deployment needs. This project aims to simplify online shopping by offering a robust and efficient system that benefits both customers and store administrators.
